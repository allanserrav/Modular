Contexto core dos recursos angular no projeto webcore. 
TopMenu
	- Este item será informado em multiplos pagina.decorator no parametro categoria da pagina. Cada categoria terá vários nomes de pagina (submenu).
SubMenu
	- Este item será informado no pagina.decorator em nome da página.

Header
	- Fica acima do menu
	- Aqui pode mostrar informações da sessão do usuário, versão do aplicativo (serviço)
	- Aplicativo que está em uso
		- Aplicativo de estoque
		- Aplicativo de venda

Engine de pagina
	Form
	Fornece toda a estrutura para submeter uma pagina. Será fornecido atráves de ng-content.
